## python中对象的三个特征
### 身份
- 对象在内存中的地址,可以通过id去查看
### 类型
- int类型、字符串类型...
### 值
`a = 1`, 1是指这个对象的值, 1会被python的int类型进行封装, 然后使用a指向1这个对象
