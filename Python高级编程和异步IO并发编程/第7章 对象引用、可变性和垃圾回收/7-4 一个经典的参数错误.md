## 经典参数传递错误
### 传入int值
```python
def add(a,b):
    a+=b
    return a
a=1
b=2
c=add(a,b)

print(a,b,c)
>>> 1 2 3
```

### 传入list
```python
def add(a,b):
    a+=b
    return a
a=[1]
b=[2]
c=add(a,b)

print(a,b,c)
>>> [1, 2] [2] [1, 2]
```
内部使用的是+=, 所以内存地址不变, 所以在执行`a+=b`的前后a指向的是同一个地址, 只不过地址里面的内容发生改变.

### 传入元组
```python
def add(a,b):
    a+=b
    return a
a=(1)
b=(2)
c=add(a,b)

print(a,b,c)
>>> 1 2 3
```
