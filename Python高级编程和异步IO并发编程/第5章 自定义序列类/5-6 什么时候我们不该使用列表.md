## array
前面在序列分类的时候提到了一种array类型, 它其实是C语言中的数组, 与list主要有如下区别:
- array只能存放指定的数据类型
- array的数据在内存中是连续的
- array的效率和性能会比list更高

```python
import array

my_array = array.array("i")
my_array.append(1)
my_array.append(2)

print(my_array)

>>> array('i', [1, 2])
```
